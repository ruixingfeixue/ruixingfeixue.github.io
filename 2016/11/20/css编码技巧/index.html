css编码技巧

尽量减少代码重复

当某些值相互依赖时，应该把它们的相互关系用代码表达出来。如：

font-size: 20px;line-height: 30px;
font-size: 20px;line-height: 1.5;

做按钮时应考虑到底哪些效果应该跟着按钮一起放大，而哪些效果是保持不变的。

Q：怎样解决不能颜色背景的按钮？(Demo点这里)
A：把半透明的黑色或白色叠加在主色调上，即可产生主色调的亮色和暗色变体。
说人话，加入透明色。

代码变得更DRY的方法

代码易维护 vs. 代码量少
代码易维护和代码量少不可兼得，应该根据实际情况取舍

currentColor
CSS中有史以来的第一个变量,会自动地从文本颜色那里得到颜色。

合理使用简写
我应该使用预处理器吗

如果使用得当，它们在大型项目中可以让代码更加灵活
CSS 的文件体积和复杂度可能会失控。
调试难度会增加，因为你在开发工具中看到的CSS代码并不是你写的源代码。
预处理器在开发过程中引入了一定程度的延时。
很多受预处理器启发的特性都已经以各种方式融入到原生CSS中了。
